## 1. AV コンテンツ制作パイプライン

### 1.1 業界構造: 3つの主要プレイヤー

| プレイヤー | 役割 | 例 |
|---|---|---|
| **メーカー（版元）** | 企画立案、予算管理、販売・流通、ブランド管理 | S1 NO.1 STYLE, MOODYZ, PRESTIGE, SOD |
| **プロダクション** | タレントマネジメント、キャスティング営業 | T-POWERS, MARKS JAPAN, C-MORE |
| **制作会社** | 撮影・編集の実務（監督・カメラ・照明・音声・編集） | フリーランスチーム中心 |

**資金の流れ**: メーカーが全予算を一括管理し、監督・制作会社・プロダクションへのギャランティは全てメーカーから支払われる。プロダクションはメーカーに対して営業し、自社所属の出演者のプロフィールを持ち込む。

### 1.2 制作工程フロー

```
企画 → キャスティング → 契約締結（AV新法対応: 撮影1ヶ月前まで）
  → 撮影（通常1-3日） → 編集（1-2週間）
    → 倫理審査（映像倫/JCRC）
      → 審査通過 → 配信・販売（撮影完了4ヶ月後以降）
        → FANZA/DVDメーカー/EC
```

### 1.3 制作費の内訳

| 項目 | 金額レンジ | 備考 |
|---|---|---|
| **制作費総額（企画もの）** | 100万〜150万円 | 出演料別 |
| **制作費総額（単体もの）** | 200万〜500万円+ | トップ女優起用時 |
| **出演料: 企画単体** | 5万〜15万円/本 | 経験・知名度による |
| **出演料: 専属単体** | 20万〜400万円/本 | トップ層は100万円超 |
| **監督** | 15万〜30万円/本 | フリーランスが主流 |
| **カメラ・照明・音声** | 各5万〜15万円/日 | 制作会社経由 |
| **編集** | 10万〜20万円/本 | ポストプロダクション |
| **スタジオ・ロケ費** | 5万〜30万円/日 | 都内スタジオ基準 |

**歴史的推移**: 2000年頃は1本あたり1,000万円近い制作費だったが、海賊版の蔓延により制作費は年々低下。現在は100万〜150万円が企画ものの標準的予算。

### 1.4 コンテンツの収益構造

```
DVD販売価格 3,000円の場合:
  ├── FANZA取り分: 2,100円（70%）
  ├── メーカー取り分: 900円（30%）
  │     ├── 制作費回収
  │     ├── 出演料支払い（プロダクション経由）
  │     ├── 審査費用
  │     └── メーカー利益
  └── ※大手メーカーで30%、中小は10-20%のケースも
```

デジタル配信の場合、FANZAの手数料は売上の概ね40%（公称は卸価格80%=手数料20%だが、クーポン・値引き等の実質負担を含めると40%前後）。ただし中小メーカーほど条件は厳しくなる。

## 2. 物理商品（アダルトグッズ）製造サプライチェーン

### 2.1 グローバル製造構造

**世界の性玩具の70%以上が中国で製造**されている。主要製造拠点:

| 地域 | 特徴 | 主要企業例 |
|---|---|---|
| **深圳（Shenzhen）** | IoTバイブレーター、電子部品統合 | WINYI, Lovense（設計）, ALWUP |
| **東莞（Dongguan）** | シリコン成型、大量生産 | 多数のOEM工場 |
| **日本国内** | 高品質ブランド製品、素材研究 | TENGA（東京本社）, TOMAX |

### 2.2 製造工程（バイブレーター例）

```
素材調達 → シリコン成型 → 電子部品組込 → 品質検査 → パッケージング → 出荷
  │            │              │             │
  │            │              │             ├── 耐久試験
  │            │              │             ├── 素材安全性試験
  │            │              │             └── 動作確認
  │            │              │
  │            │              ├── モーター
  │            │              ├── バッテリー
  │            │              └── Bluetooth/IoTモジュール（スマート製品の場合）
  │            │
  │            ├── プラチナ硬化シリコン（最高品質）
  │            ├── 医療グレードシリコン
  │            └── TPE（廉価品）
  │
  ├── シリコン原料
  ├── ABS樹脂
  └── 電子部品（深圳調達）
```

### 2.3 安全基準・認証

| 認証 | 対象 | 内容 |
|---|---|---|
| **CE マーキング** | EU市場 | 一般製品安全指令準拠、電子部品含む場合は低電圧指令も |
| **RoHS** | EU市場 | 有害物質（鉛・水銀・カドミウム等）制限 |
| **REACH** | EU市場 | 化学物質安全性 |
| **FDA 21 CFR Part 177** | 米国市場 | 食品接触グレード/医療グレードシリコン |
| **ISO 10993** | 国際 | 生体適合性試験 |
| **USP Class VI** | 国際 | 医療グレードシリコン認証 |
| **PSE** | 日本市場 | 充電式製品の電気安全法準拠 |

**重要**: 性玩具は多くの国で「医療機器」には分類されないため、FDAの事前承認は不要。ただし医療・治療効果を謳う場合はFDA規制対象となる。

### 2.4 日本国内流通チャネル

```
製造（中国OEM/国内工場）
  │
  ├── 卸業者（NLS = Night Life Style 等）
  │     ├── 実店舗（アダルトショップ、信長書店等）
  │     ├── EC（NLS通販、通販大魔王等）
  │     └── ドン・キホーテ（一般流通）
  │
  ├── Amazon Japan（Adult カテゴリ承認必要）
  │     └── FBA対応可、ただし広告制限あり
  │
  ├── 楽天市場（規約内のみ）
  │
  └── ブランド直販（TENGA公式、LC公式等）
```

## 3. 法的権利構造・規制フレームワーク

### 3.1 AV出演被害防止・救済法（AV新法, 2022年6月施行）

#### 背景
成人年齢引下げ（20歳→18歳）に伴い、18-19歳が契約弱者としてAV出演被害に遭うリスクへの対応。

#### 主要規定

| 規定 | 内容 |
|---|---|
| **契約前説明義務** | 出演内容の事前書面説明、契約書の交付必須 |
| **撮影開始制限** | 契約締結から1ヶ月間は撮影不可 |
| **公表制限** | 撮影完了から4ヶ月間は作品公表不可 |
| **無条件解除権** | 年齢を問わず、撮影完了後1年間は無条件で契約解除可能 |
| **罰則** | 違反には罰金・懲役の可能性 |

#### 施行後の状況
2022年6月〜10月の間に自治体への相談103件（契約前70件、契約後12件）。業界側からは「制作期間の長期化によるコスト増」「中小メーカーの経営圧迫」との声もあり、出演者100名超が法改正を求めるデモを実施。

### 3.2 風営法: 映像送信型性風俗特殊営業

#### 対象
「専ら性的好奇心をそそるため、性的な行為を表す場面又は衣服を脱いだ人の姿態の映像を、電気通信設備を用いて客に伝達する営業」

#### 届出要件
- 営業開始**10日前**までに所轄警察署（生活安全課）に届出
- 必要書類: 開始届出書、住民票/登記簿謄本、営業概要書、誓約書、賃貸借契約書等
- 海外プラットフォーム利用でも、日本国内で営業する限り届出必須
- **2024年7月**: myFansが「クリエイター個人単位での届出義務化」を発表

#### 罰則
届出なし営業: **6ヶ月以下の懲役 or 100万円以下の罰金 or 併科**

### 3.3 倫理審査機関

| 機関 | 正式名称 | 審査対象 | 備考 |
|---|---|---|---|
| **JCRC** | 一般社団法人日本コンテンツ審査センター | 映像ソフト（ビデオ/DVD）、アダルトゲーム | 2016年に映像倫+東日本CS+全日本CSが統合 |
| **ソフ倫（EOCS）** | コンピュータソフトウェア倫理機構 | PCゲーム中心 | 審査非通過ソフトの流通を業界全体で制限 |

#### 審査プロセス
1. 1作品に対し複数審査員が独立審査
2. 各審査結果を基に最終判断審査員が映像確認
3. 判断困難な作品は複数審査員で協議
4. なお結論が出ない場合は外部有識者を交えて検討

#### 審査基準の核心
- **わいせつ物頒布等の罪（刑法175条）** 回避が最大の目的
- モザイク処理の基準（サイズ・範囲）が実質的に最重要
- 審査不通過の場合、FANZA等主要プラットフォームでの配信不可

### 3.4 著作権・肖像権・パブリシティ権

| 権利 | 帰属 | 実務上の扱い |
|---|---|---|
| **著作権** | メーカー（映画の著作物として） | 出演者には帰属しない |
| **著作隣接権（実演家の権利）** | 出演者に発生するが、契約で譲渡/許諾が一般的 | AV新法で無条件解除権が上書き |
| **肖像権** | 出演者固有 | 契約で使用許諾、解除権行使で復活 |
| **パブリシティ権** | 出演者固有 | 名前・写真の商業利用に関する権利 |

### 3.5 コンプライアンスコスト

業界全体で**売上高の22%**（2024年）がコンプライアンス費用に充てられている。年齢確認、同意記録、審査費用、法務費用、決済規制対応等を含む。

## 4. 決済インフラ

### 4.1 クレジットカード決済の構造的問題

#### MCC（加盟店カテゴリコード）

| MCC | 分類 | リスクレベル |
|---|---|---|
| **5967** | ダイレクトマーケティング（インバウンドテレマーケティング） | 高リスク |
| **7841** | ビデオレンタル | 高リスク |
| **5993** | 成人向け製品 | 高リスク |

#### VISA/Mastercard規制の詳細

**Mastercard BRAM（Business Risk Assessment and Mitigation）プログラム**:
- 2021年10月15日から施行
- 非対面取引の成人向けコンテンツ・サービス事業者に適用
- **必須要件**:
  - 全コンテンツの年齢確認（出演者・利用者双方）
  - 同意の文書化・記録保持
  - コンテンツレビュープロセスの導入
  - 違法コンテンツの報告・削除メカニズム
  - 第三者によるコンプライアンス監査

**2024年4月改定**:
- 高リスク加盟店の登録料: $500 → **$950/プロバイダー/アクワイアラー**
- 取引手数料: **10セント/取引 + 10bps/取引額**

#### チャージバック率

| 区分 | チャージバック率 |
|---|---|
| **全業界平均** | 0.60% |
| **アダルト業界** | 1.5%〜3.0%（全業界平均の2.5〜5倍） |
| **許容上限** | 1.0%（超過でアカウント凍結・解約リスク） |

### 4.2 加盟店手数料の比較

| 区分 | 手数料率 |
|---|---|
| **一般加盟店** | 1.5%〜3.5% |
| **高リスク加盟店（一般）** | 3.49%〜3.95% + $0.25/件 |
| **アダルト加盟店** | **3.5%〜6.5%** |
| **+リザーブ（保証金）** | 売上の5%〜10%を6ヶ月間保持 |

### 4.3 代替決済手段

#### 暗号通貨

暗号通貨はアダルト業界にとって構造的に有利:

| 利点 | 詳細 |
|---|---|
| **チャージバック不可** | ブロックチェーン確認後は取消不能 |
| **プライバシー** | 利用者の明細に記載されない |
| **低手数料** | 従来の3-10%に対し、1%未満のケースも |
| **グローバル対応** | 国際取引の障壁なし |
| **検閲耐性** | カード会社の恣意的な取引停止リスク回避 |

**主要プロバイダー**:

| サービス | 対応通貨数 | 特徴 |
|---|---|---|
| **NOWPayments** | 150+ | KYC不要、即時ペイアウト |
| **CoinGate** | 70+ | 法定通貨（USD/EUR/GBP）も対応 |
| **BTCPay Server** | BTC中心 | Lightning Network対応、セルフホスト |
| **ATLOS** | 主要通貨 | 非カストディアル、No-KYC |

#### その他の代替手段（日本市場）
- **キャリア決済**: docomo/au/SoftBank
- **コンビニ決済**: 匿名性が高い
- **プリペイドカード**: DMMポイントカード等
- **PayPay/LINE Pay**: 一部プラットフォームで対応
- **銀行振込**: 法人取引の基本

### 4.4 日本特有の決済課題

FANZAが圧倒的なシェアを持つ理由の一つが、DMMグループが自社で決済インフラを保有していること。独立系メーカーやクリエイターが独自に決済を構築する場合、高リスク加盟店審査のハードルが極めて高い。

## 6. 景気耐性の検証

### 6.1 「不況に強い」説の実態

**結論: 部分的に正しいが、セグメントによって大きく異なる。**

#### デジタルコンテンツ（強い耐性）

| 時期 | 状況 |
|---|---|
| **2008年金融危機** | 無料ポルノサイトへの移行が加速。Pornhubなどのチューブサイトが急成長。有料DVDは打撃を受けたが、デジタル全体のトラフィックは増加 |
| **COVID-19（2020年3月）** | Pornhubが有料コンテンツを無料開放 → 日平均115M → **160M訪問者**（+39%）。地域別で38-61%のトラフィック増 |
| **COVID-19中長期** | 初期スパイク後は正常化。2020年5月: 36%が視聴 → 8月: 22% → 10月: 21%と減少傾向 |

#### 物理サービス（脆弱）

| セグメント | 不況時の影響 |
|---|---|
| ストリップクラブ | ラスベガスで**-12%**の売上減少 |
| 合法売春施設 | ネバダ州で**-20%**の売上減少 |
| セックスワーカー個人 | トップパフォーマーでも前年比**-50%**の収入減少 |

#### 物理商品（中程度の耐性）

性玩具市場はCOVID-19中に「巣ごもり需要」で成長。LovehoneyはCOVID期間中に売上急増を報告。TENGA もYoY 110%の成長を維持。

### 6.2 セグメント別景気耐性マトリクス

| セグメント | 耐性 | 根拠 |
|---|---|---|
| 無料デジタルコンテンツ（広告収入） | ★★★★★ | 不況時にむしろトラフィック増 |
| 有料デジタル配信 | ★★★★☆ | 低価格帯は耐性あり、高額サブスクは影響 |
| クリエイタープラットフォーム | ★★★★☆ | 失業→新規クリエイター流入、ファン側は節約傾向 |

| 風俗サービス | ★★☆☆☆ | 裁量的支出として真っ先に削減 |
| ライブイベント/店舗 | ★★☆☆☆ | 外出控え＋節約の直撃 |

### 6.3 真の耐性メカニズム

アダルト産業が全体として一定の耐性を持つ理由:
1. **基本的欲求**: 性的欲求は景気に左右されない
2. **代替効果**: 高額サービス → 低コストのデジタルコンテンツへシフト
3. **デジタル限界費用ゼロ**: 制作済コンテンツの追加配信コストがほぼゼロ
4. **匿名性**: 不況時の精神的ストレス→消費、かつ他の娯楽より隠れた消費が可能
5. **クリエイター供給増**: 不況時に副業・新規参入が増加

ただし「完全に不況に強い」わけではなく、**デジタルへの代替が進む**という形で耐性を発揮する。物理的サービスは明確に打撃を受ける。

### 7.5 2025年の大きな規制変化

**シアリスがOTC化**: 2025年9月、日本の厚生労働省審議会がシアリスの一般用医薬品（OTC）としての販売を承認。
- **店頭販売**: 薬剤師の説明を条件に薬局で購入可能に
- **オンライン販売**: 2026年5月以降、薬剤師の説明を条件にオンライン購入も可能に
- **日本初のOTC-ED治療薬**として歴史的な規制緩和

これにより:
1. 受診ハードルの撤廃 → 潜在患者層の顕在化
2. 価格競争の激化 → ジェネリック＋OTCで更なる低価格化
3. オンラインクリニックの優位性低下（処方不要になるため）
4. ドラッグストアチェーン（マツキヨ、ウエルシア等）が新規参入

### 7.6 オンラインクリニックの成長

オンラインED診療の流れ:
1. Webで問診票提出
2. 医師によるオンライン診察（ビデオ/チャット）
3. 処方箋発行
4. 提携薬局から自宅配送（最短翌日）

**DMMオンラインクリニック**はDMMグループの一部として、FANZAユーザーベースとのシナジーを持つ独特のポジショニング。

## 補足: 産業横断的な構造的特徴

### 規制の二面性

アダルト産業の規制は**参入障壁**であると同時に**既存プレイヤーの堀（moat）**でもある:
- 高い決済コスト → 資本力のあるプレイヤーが有利
- 倫理審査の必須性 → 審査機関との関係構築が参入障壁
- 風営法届出 → 法務コストが零細事業者を排除
- AV新法の制作期間制約 → 資金繰り余力が必要

### デジタルシフトの不可逆性

| 年代 | 主要媒体 | 収益構造 |
|---|---|---|
| 〜2000年 | VHS/DVD | 物理流通、メーカー主導 |
| 2000〜2010年 | DVD + 初期配信 | FANZAの台頭 |
| 2010〜2020年 | デジタル配信主流 | FANZA独占化、クリエイターエコノミー萌芽 |
| 2020年〜 | クリエイタープラットフォーム | OnlyFans/myFans、個人の直接収益化 |

**現在のトレンド**: メーカー経由の従来型制作から、クリエイター個人がプラットフォーム上で直接収益化する**D2C（Direct-to-Consumer）モデル**への移行が加速。
